# Threat-matrix
## Team Project

This project was developed as a collaborative academic cybersecurity project by a team of students, with responsibilities distributed across frontend development, backend implementation, and security monitoring components.

### My Contributions
- Designed and implemented threshold-based threat detection logic
- Developed detection rules for:
  - Brute Force Attacks
  - DDoS Activity
  - Port Scanning
  - Ping Sweep Detection
- Performed log analysis and threat classification
- Contributed to backend security functionality
- Assisted in SOC monitoring and alert generation workflows

### Team Contributions
- Frontend development and user interface design
- Dashboard visualization and charts
- Authentication and integration components
- Testing, debugging, and project deployment
